0x06. C - More pointers, arrays and strings

This project is part of the low-level programming curriculum.

It focuses on practicing:
- Pointers
- Arrays
- String manipulation in C

Functions implemented in this project:

0. _strcat - concatenates two strings
1. _strncat - concatenates two strings using n bytes
2. _strncpy - copies a string
3. _strcmp - compares two strings
4. reverse_array - reverses the content of an array
5. string_toupper - converts lowercase letters to uppercase
6. cap_string - capitalizes all words of a string
7. leet - encodes a string into 1337

All programs were compiled using:
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
