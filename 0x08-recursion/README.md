# 0x08. Recursion

## Description
This project focuses on learning recursion in C. You will implement several functions using **recursion only**, without any loops.  
All functions use `_putchar` for output and follow **Betty style** guidelines.

## Files
- `_putchar.c` – prints a single character to stdout.
- `0-puts_recursion.c` – prints a string followed by a new line using recursion.
- `main.h` – header file containing function prototypes.
- `0-main.c` – test file to check the implementation of `_puts_recursion`.

## Compilation
To compile the files, run:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 _putchar.c 0-main.c 0-puts_recursion.c -o 0-puts_recursion
