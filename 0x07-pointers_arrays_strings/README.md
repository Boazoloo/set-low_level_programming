# 0x07. C - Even More Pointers, Arrays and Strings

This project focuses on advanced pointer manipulation, memory functions,
and string handling in C. You will practice working with:

- Double pointers
- 2D arrays
- Recreating C standard library string functions manually

## Tasks

0. _memset
1. _memcpy
2. _strchr
3. _strspn
4. _strpbrk
5. _strstr
6. print_chessboard
7. print_diagsums

## Requirements

- Use C - Functions Betty style
- No global variables
- Maximum 5 functions per file
- Use only `_putchar` for output, do not push `_putchar.c`
- All prototypes must be in `main.h`
